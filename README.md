## 📜 Getting Started
### Clone the repo.

```bash
git clone https://github.com/gavinheiner/personal-portfolio.git
```

### Install dependencies.
```bash
npm install
# or
yarn install
```
### Run locally.

```bash
npm run dev
# or
yarn dev
```

### Build for production.

```bash
npm run build
# or
yarn build
```
